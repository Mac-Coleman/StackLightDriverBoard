# Stack Light Driver Board

This repository hosts my KiCad project.

Currently there is only a schematic.

Footprints and 3D models of most components are provided by [CDFER's JLCPCB component library for KiCad](https://github.com/CDFER/JLCPCB-Kicad-Library).


## Footprints and 3D models of the following components provided by SnapMagic:
* TSR_1-2433E
* 282837-2
* B2B-XH-A
* ESP32-S3-WROOM-1-N4
* PJ-102AH

## 3D models provided by Adafruit:
